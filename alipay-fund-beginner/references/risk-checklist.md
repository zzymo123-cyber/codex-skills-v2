# Risk Checklist

Use this checklist to review the user's fund holdings conservatively.

## Suitability

- Is the money needed within 1 year? If yes, high-volatility funds are usually a mismatch.
- Does the user lack emergency cash? If yes, prioritize cash buffer before adding fund risk.
- Can the user emotionally accept the likely drawdown? If not, reduce risk language and avoid encouraging additions.
- Is the user using profit/loss as the main reason to act? Flag this as behavioral risk.

## Portfolio Structure

- Too much in equity, industry, theme, overseas, or commodity funds for the user's horizon.
- Too many funds with similar names, indexes, managers, sectors, or top holdings.
- One fund or one theme dominates the portfolio.
- The user owns many funds but the underlying exposure is not truly diversified.
- The user treats bond funds, fixed-income-plus funds, or low-risk labels as guaranteed stable.

## Behavior

- Recent high returns are the main reason for buying.
- Recent losses are the main reason for adding money to "回本".
- The user checks daily and wants frequent action.
- The user wants a simple answer before understanding what the fund owns.
- The user asks whether today is the right buying point.

## Labeling Guide

- `继续观察`: Risk is understandable, allocation is not obviously excessive, and no urgent action is needed.
- `减少新增`: Existing holding may be acceptable, but adding more would increase concentration or volatility.
- `不建议加仓`: Risk, concentration, time horizon, or emotional state makes additional buying unsuitable.
- `先学习再决定`: The user does not understand the type, risk, or reason for holding.
- `需要补充信息`: Fund code, holding percentage, horizon, risk tolerance, or screenshot details are missing.
